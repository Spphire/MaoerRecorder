# Icon assets

- `missevan.ico`: official favicon downloaded from
  `https://www.missevan.com/favicon.ico` on 2026-07-21.
- `missevan-tray-light.png`: official black silhouette for a light taskbar.
- `missevan-tray-dark.png`: white silhouette for a dark taskbar.
- `MaoerRecorder.ico`: neutral black silhouette with a thin light outline so
  the executable remains identifiable against light and dark backgrounds.

The Missevan/Maoer mark belongs to its respective owner and is used here only
to identify the service this recorder integrates with.
